------------


**运行方式：运行Main.java，依次输入道路网格尺寸和联通指令即可看到效果。**

**Invalid number format.测试：**

| 输入 | f 3 |
|--|--|
| 输出 | Invalid number format. |

------------


**Number out of range.测试：**

| 输入 | -1 1 |
|--|--|
| 输出 |Number out of range. |

------------


**Incorrect command format.测试：**

|  输入 |  3 3  |
| ------------ | ------------ |

------------

| 输入  | 3 3  |
